# S3 storage configuration

To use this storage install extension with `s3` extras.

```sh
pip install 'ckanext-files[s3]'
```


```ini
## Storage adapter used by the storage
ckanext.files.storage.NAME.type = files:s3
## Name of the S3 bucket where uploaded data will be stored.
ckanext.files.storage.NAME.bucket =
## The access key for AWS account.
ckanext.files.storage.NAME.key =
## The secret key for AWS account.
ckanext.files.storage.NAME.secret =
## Custom AWS endpoint.
ckanext.files.storage.NAME.endpoint =
## The AWS Region used in instantiating the client.
ckanext.files.storage.NAME.region =
```
